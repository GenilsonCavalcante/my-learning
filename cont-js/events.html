Events

Eventos são interações que ocorrem em ações geradas pelos usuários, ou simplesmente podem ser programadas para ocorrem.

Podemos citar exemplos de eventos como:

  • Butão criado
  • Mudança de cor de fundo de acordo com o horário do sistema
  • Mudança de imagens, quando o usuário passa o mouse por cima dela
  • Carregamento de uma página
  • Quando o usuário arrasta o mouse para sair do site e abir uma janela de aviso

A lista de eventos é gingantesca. Com o javascript, ele nos proporciona a executar códigos quando eventos são detectados.

Veje mais tipos de eventos em javascript:

  • onchange - Um elemento HTML foi alterado
  • onclick - O usuário clica em um elemento HTML
  • onmouseover - O usuário move o mouse sobre um elemento HTML
  • onmouseout - O usuário move o mouse para longe de um elemento HTML
  • onkeydown - O usuário pressiona uma tecla do teclado
  • onload - O navegador terminou de carregar a página
