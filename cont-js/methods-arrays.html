Methods-Arrays

Dentro dos arrays, temos alguns métodos que possibilitam coisas essenciais para podermos trabalharmos com eles, Vejamos alguns métodos:

Método .toString(). Esse método converte um array em uma lista dos valores formatados separados por uma vírgula.

                    const fruits = ["Banana", "Orange", "Apple"];
                    document.getElementById("demo").innerHTML = fruits.toString();

                    //Resulta em: Banana, Orange, Apple

Temos ainda um método bem similar ao .toString(), o método .join(), seu diferencial é que podemos formatar o tipo de separador, podendo ser uma vírgula, asterísco, ou qualquer outro caractere.

                    const fruits = ["Banana", "Orange", "Apple"];
                    document.getElementById("demo").innerHTML = fruits.join(" * ");

                    //Resulta em: Banana * Orange * Apple

O método .pop(), remove o último elemento.

O método .push, adiciona um elemento ao final de uma matriz

O método .shift(), remove o primeiro elemento da matriz e readapta os elementos posteriores a os índices inferiores, começando do índice 0 e continuando a sequência.

O método .unshift(), adiciona um elemento ao início, e faz a relocação dos índices devidamente em suas posições, adaptando a o padrão corretamente.

Uma boa prática de adicionar um item ao último elemento de um array, é através do tamanho do array calculado com a propriedade .length. Veja o exemplo:

                    const fruits = ["Banana", "Orange", "Apple"]
                    fruits[fruits.length] = "Kiwi";

                    //Resulta em um array com Banana, Orange, Apple, Kiwi

Estes foram alguns dos métodos, ainda existem muitos, muitos até específicos.
