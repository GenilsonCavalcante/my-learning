Arrays

Arrays, listas, matrizes, como preferir chamar. São nomes diferentes que representa a mesma coisa. Um array é basicamente uma lista de valores que podemos atribuir a uma única variável.

Exemplo:

                    let car = ["Ferrari", "Chevet", "Fusca"];
                    //Declaração de um array!

Isso acima é o mesmo do que isso abaixo:

                    let car1 = "Ferrari";
                    let car2 = "Chevet";
                    let car3 = "Fusca";

A vantagem de se utilizar matrizes é que se referirmos a listas grandes, para percorrer um a um dessa segunda forma demostrada, isso dá um grande trabalho. Mas se usarmos um array, isso pode ser facilitado e muito, pois com uma matriz, podemos recorrer os valores através de seus índices.

Note que as matrizes utilizam os seus índices para acessar os seus valores, e que os objetos, conteúdo visto anteriormente, usa-se de nomes "atributos" para acessar os seus valores.

Como dito antes, acessamos os valores de uma array pelo seu índice.

                    const fruits = ["pera", "maça", "banana"];
                    fruits.[0] //return "pera"

Uma das melhores formas de percorrer um array é através de um for:

                    const fruits = ["Banana", "Orange", "Apple", "Mango"];
                    let fLen = fruits.length;

                    text = "<ul>";
                    for (let i = 0; i ≤ fLen; i++) {
                      text += "<li>" + fruits[i] + "</li>";
                    }
                    text += "</ul>";

Existe muitas outras formas mais específicas de se percorrer a uma array.

Podemos adicionar elementos a um array através do .push

                    const fruits = ["Banana", "Orange", "Apple"];
                    fruits.push("Lemon");  // Adds a new element (Lemon) to fruits
