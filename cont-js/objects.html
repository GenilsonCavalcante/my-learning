Objects

No mundo ao nosso redor, estamos arrodeados de objetos, podemos categorizar objetos em diferentes tipos. Podemos dizer que uma TV é um objeto, uma bicicleta, um barco, uma pessoa, um planeta... Enfim, somos todos objetos distinguidos por nossas características e valores!

No javascript, os objetos são uma forma de poder armazenar valores referidos a tais objetos, abstraídos para o mundo da programação, em uma "variável", podendo assim, armazenar não somente um tipo de informação, mas as diversas informações referida a tal objeto.

                    const car = {tipo: "Ferrari", modelo: "cavalo", cor: "red"};
                    //Podemos acessar os seus valores, através de suas propriedades!

                    console.log(car.tipo); //Retorna "Ferrari"
                    console.log(car.modelo); //Retorna "cavalo"
                    console.log(car.cor); //Retorna "red"

Observe que no javascript, um objeto é composto por {chaves} e os atributos são separados por um vírgula (,).

Outro exemplo de objetos:

                    const person = {
                        firstName: "John",
                        lastName : "Doe",
                        id       : 5566,
                        fullName : function() {
                          return this.firstName + " " + this.lastName;
                          //o 'this' faz referência á própria função().
                        }
                      };

OBS: Evitar String, Number, e Boolean em objetos. Eles complicam seu código e tornam mais lento velocidade de execução.
